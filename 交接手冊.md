## 一、專案概述

**JoinUp!** 是一個專為暨大學生設計的校園互助平台，採用 PWA（Progressive Web App）技術開發。學生可以透過這個平台找室友、揪團活動、共乘等等。

**目前開發進度**：

## 二、交接流程

### 第一步：下載專案

1. 打開電腦的終端機（Windows 用 PowerShell，Mac 用 Terminal）
2. 輸入以下指令下載專案到桌面：
   - Windows：`cd $HOME\Desktop`
   - Mac：`cd ~/Desktop`
3. 輸入：`git clone https://github.com/Delia1203/Joinup.git`
4. 下載完成後，你的桌面會出現一個叫 `Joinup` 的資料夾

### 第二步：用 Antigravity 開啟專案

1. 打開 Google AI Studio（Antigravity 的平台）
2. 選擇「開啟資料夾」，選擇剛才下載的 `Joinup` 資料夾
3. 對 Antigravity 說：「我是新接手這個專案的開發者，請幫我介紹一下專案結構」
4. Antigravity 會告訴你專案的架構和已完成的功能

### 第三步：測試專案是否能運作

1. 對 Antigravity 說：「幫我啟動本地伺服器，我要測試這個網站」
2. 打開瀏覽器，輸入 `http://localhost:8000`
3. 試著註冊一個測試帳號（用 `test@ncnu.edu.tw` 這種格式）
4. 試著發布一個租屋貼文或發起一個活動
5. 如果都能正常運作，代表設置成功

### 第四步：接手網站部署權限

這個專案使用 Vercel 來部署網站到網路上。你需要：

1. 去 vercel.com 註冊或登入帳號
2. 請前一位開發者把你加入專案的協作者
3. 或者你也可以自己重新部署（對 Antigravity 說：「幫我部署到 Vercel」）

## 三、日常開發方式

### 用 Antigravity 開發新功能

這個專案最大的特色就是用 AI 輔助開發，所以你不需要自己寫太多程式碼。

**開發步驟**：

1. **告訴 Antigravity 你要做什麼**
   
   例如你想新增「一起運動吧」功能，可以這樣說：
   > 「我要新增一起運動吧的功能，請參考一起出去吧（travel.js）的邏輯。需要在首頁加一個按鈕，建立新的頁面，讓學生可以發起運動揪團。」

2. **Antigravity 會自動幫你**：
   - 建立需要的檔案
   - 寫好程式碼
   - 告訴你改了哪些地方

3. **測試功能**
   
   對 Antigravity 說：「幫我啟動伺服器測試」，然後在瀏覽器看看新功能是否正常運作。

4. **存檔到 Git**
   
   測試沒問題後，對 Antigravity 說：「幫我把這次的修改存到 Git，commit 訊息寫：新增一起運動吧功能」

5. **部署到網路上**
   
   只要你把檔案存到 Git，Vercel 就會自動把網站更新到網路上，通常 1-2 分鐘就完成了。

## 四、專案架構說明

你不需要記住所有檔案，只要知道這 **3 個最重要的檔案**：

1. **`js/views/home.js`** - 首頁
   - 想加新功能的按鈕，就改這個檔案

2. **`js/models/mockStore.js`** - 資料庫
   - 所有資料都存在這裡（目前用瀏覽器的 LocalStorage 模擬資料庫）

3. **`js/app.js`** - 路由設定
   - 新增頁面時要在這裡設定路徑

**參考範例**：
- 如果要做「即時加入」的功能（像運動、讀書會），參考 `travel.js`
- 如果要做「申請審核」的功能（像租屋、共乘），參考 `groupbuy.js`

## 五、常見問題處理

### 情況一：改了程式碼但網頁沒變化

**解決方法**：按 F12 打開開發者工具 → 找到 Application → Service Workers → 點 Unregister → 重新整理頁面

### 情況二：Git 上傳失敗

**解決方法**：對 Antigravity 說「Git push 失敗了，幫我解決」

### 情況三：Vercel 部署失敗

**解決方法**：去 Vercel 網站的 Dashboard 看錯誤訊息，通常 Antigravity 可以幫你解決

### 情況四：不知道怎麼用 Antigravity

**解決方法**：
- ✅ 好的問法：「我要新增 XXX 功能，請參考 YYY 的邏輯，需要做 1. 2. 3.」
- ❌ 不好的問法：「幫我做一個功能」（太模糊了）
